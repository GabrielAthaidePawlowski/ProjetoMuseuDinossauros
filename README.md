# DinoMuseum API 🦖

Projeto semestral para a disciplina de Arquitetura de Aplicações Web (2026.1). 
Um sistema de documentação e catálogo de dinossauros estilo museu.

## 🛠 Stack Prevista
- **Backend:** .NET 10 (C#)
- **Banco de Dados:** MongoDB
- **Frontend:** React ou Vanilla JS (Navegação Assíncrona)

## 📋 Entidades Principais
1. **Dinosaur:** Nome, espécie, dieta, período.
2. **Era/Habitat:** Nome da era geológica e descrição.
